# Cyril Analytics

A structured self-study portfolio documenting my journey from SQL foundations to a full data analytics workflow.

I am a third-year BSc Mathematics and Computer Science student at Multimedia University of Kenya (graduating October 2027), building practical data skills alongside my degree — one day at a time, tracked publicly.

---

## What This Repository Contains

Daily SQL and Python work organised by week. Each day covers a concept, exercises, and a small project. Everything is written, run, and pushed from the terminal.

**Current stack:** SQLite · Python 3.12 · pandas (upcoming) · matplotlib (upcoming) · Git

---

## Roadmap Overview

| Phase | Days | Topics |
|---|---|---|
| SQL Foundations | 1 – 3 | SELECT, JOINs, subqueries, CTEs, INSERT, UPDATE, DELETE, triggers, window functions |
| SQL + Python Basics | 4 – 6 | Views, indexes, Python variables, lists, loops, dictionaries, functions, SQLite from Python |
| Data Analysis | 7 – 9 | pandas, data manipulation, matplotlib visualisations |
| Full Pipeline | 10 | Database → Python → analysis → chart |
